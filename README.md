# Arch Decman Config

Config files for [decman](https://github.com/kiviktnm/decman)

To install decman
```sh
sudo pacman -S git base-devel
git clone https://aur.archlinux.org/decman.git
cd decman/
makepkg -sic

```

To install config
```sh
git clone git@github.com:DPR1604/arch-decman-config.git
sudo decman --source arch-decman-config/source.py
```
